## Laundry Service Booking Application
3 Project Overview
The Laundry Service Booking Application is a web-based platform that allows users to browse available laundry services, add services to a cart, view selected items, and book services online. The application provides a simple and user-friendly interface for managing laundry service bookings.


# # Features
Navigation Bar
..Home
..Services
..Contact Us
..About Us
..User Login/Logout


# Browse Services
View available laundry services.
Display service image, name, and price.
Skip to the next service using the Skip Item button.
Add Items
Add selected services to the booking cart.
Display selected services in the Added Items table.
Show service name and price.


# Total Amount Calculation
Automatically updates the total amount when services are added.
Displays the final booking amount.

# Booking Form
....Full Name
....Email ID
....Phone Number
....Book Now button
....Dynamic Functionality
....Service card updates dynamically.
....Cart updates using JavaScript DOM manipulation.
....Total amount calculation in real time.

## How It Works

->User browses available services.
->Clicking Skip Item displays the next service.
->Clicking Add Item adds the selected service to the cart.
->Selected services appear in the Added Items table.
->Total amount updates automatically.
->User enters booking details.
->Clicking Book Now submits the booking.
->Future Enhancements
->Remove items from cart
->Search services
->Service category filtering
->Booking confirmation popup
->Local Storage support
->User authentication
->Responsive mobile design
->Online payment integration
